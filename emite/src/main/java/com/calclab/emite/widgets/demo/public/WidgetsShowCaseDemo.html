Welcome to emite widgets show case

Here there is some instances of emite widgets. Most of them are duplicated to view how, diferent instances the of the same widget, can be rendered in a single page.

All the widget parameters are inserted in the html (all the div attributes prefixed with 'data-') and no javascript is required for initialization.

Important: every widget div should have a unique id attribute value.
