Emite demo



  • This is only a demo of emite library and emite ui (both purelly gwt).

  • You can login with accounts like test1@emitetest.ourproject.org with password test1 (or test2, test3,... test100).

  • You can also register a jabber account in emitetest.ourproject.org (but for now only using an external jabber client).

  • With this demo you can't login using other external accounts (like your gmail account).

  • But you can use one of the test accounts for:

    • chat with other internal or external jabber accounts (as gmail accounts)

    • create and join to chat rooms in @rooms.emitetest.ourproject.org or in @conference.jabber.org or other multi-user chat capable server.

    • even, you can open to browser tabs with this demo, and chat with two different accounts in the same browser.

  • The demo accounts will be reset periodically.


More info in emite homepage.
